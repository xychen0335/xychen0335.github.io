保研经历（信息与通信工程）

2024-11-29 8 min read # 随记

该文章首发于知乎2024保研经历（信通方向）

前言

首先感谢一路上学长学姐还有各位朋友的帮助和支持。回顾这几个月，压力不可谓不大，尤其是信通这个方向可选择的院校并不多，一度让我压力拉满，幸亏最后也是达到了预期目标。

最终去向：东南大学—信息科学与工程学院（学硕）

个人保研记录见知乎2024保研记录

一、个人情况

bg：

  • 本科学校：西电
  • rk：5/142（3.5%）
  • 英语：470（CET-6）
  • 竞赛：数模国赛省一、数竞省一
  • 个人奖项：国奖*1
  • 科研/项目：基本为0

可以看出我的bg属于保研er里面一抓一大把那种，基本属于“三无选手”。在择校的时候，我主要考虑导师人品、学校title、地理位置、专业实力（有优先级），且不考虑直博。

二、前期准备

按照我的择校标准，再加上我作为一个山东人的“不远游”思想，主要将目标放在了京校和东部高校。三月份拿到排名，然后准备好文书材料，开始了一段不长不短的套磁之旅：

  • 东南：收到两封欢迎报考，与其中一位老师加了微信，东南是强com，所以主要聊了下夏令营的流程以及怎么准备
  • 北邮：bupt信通院的相当一部分老师去了智能院，所以我也是浅投了一位信通院的老师，加了微信，但没什么后续
  • 华科：HUST电通学院的名声委实不太好，只投了一位AI方向的牛导，石沉大海
  • 中科大：USTC是弱com，联系了几位六系的导师，石沉大海
  • 中科院空天院：投了两位老师，一位明确要我；另一位约了面试但当时准备不足，将我推荐给了同部门的老师
  • 中科院声学所：投了一位网络方向的牛导，但后来才得知这位老师已经不怎么招生了...

总体来说，我这一番套磁最后成效甚微，当然也与我联系的过早有关，这些基本都是我4月份联系的。后面，emm，由于一心想去东南，也就没继续套磁了。

三、保研夏令营/预推免

1.夏令营
结合对自身条件的考虑，我夏令营并没有海投，现在想想有点后悔，考虑的学校如下：

学校 入营 offer 备注
浙大工程师学院 否 /
中科大六系 否 /
东南信科（信号方向） 是 学硕 排名处于南京专硕，因学硕被鸽穿喜提学硕
西交信通 是 学硕（放弃） 点击就送
空天院 是 专硕（放弃） 建议只考虑直博
声学所 是 未参加
北邮 未报名 / 之前联系的导师问我是否参加，由于北邮夏令营不发offer，考虑预推免参加

结果基本在预料之中，不得不说华五的bar是真的高，如果不考虑直博的话， 夏令营基本上只收rk1。虽然有预期，但是得知浙大工院没入的时候还是很难过┭┮﹏┭┮

下面主要讲一下几个我参加或了解的夏令营：

  • 浙大工程师学院：只有专硕，按项目招生；我报名的是一个比较卷的项目，这也是我最终没进面的原因之一，有一些异地的项目按往年招生情况不是很卷。工院的面试很侧重考察论文、项目，这方面比较欠缺的话就有一定劣势了，这也是我没进面的原因之二。此外工院夏令营鸽子很多，预推免的机会要更大一点。
  • 中科大六系：六系和十系不一样，没有强制要求导师推荐，但是提前联系导师的优势不言而喻。bar很高，西电基本只有rk1能进，大专业会放宽一些，但还是很有难度。
  • 西交信通：西交信通是线上，bar不高，但是会卡985（陕西高校会放宽），非985的只看到了郑大，在此基础上rk前10%就能入营。夏令营活动就只有一下午的宣讲。西交信通如果要参加面试必须要联系导师并且导师向学院推荐。面试较水，主要问了研究生规划、对导师的方向有何了解以及报考意愿这些。最后参加面试的基本上全员优营+候补优营。西交老海王了，优营的效力一般，招生那边只说无需参加预推免面试。正式推免时，我推测夏令营面试考核的分数会和预推免的一起排名。不过信通作为不是很热门的学院一般都会被鸽穿...
  • 空天院：本人参加的是春令营，bar一般，但也见到了很多佬。详情可以看这我这篇经验贴空天院春令营。
  • 声学所：bar一般，按lab面试，参营之前就要确定报考的lab。面试不水，有一定通过难度。
  • 东南信号方向：学硕bar较高，专硕bar略低一些。面试分两场：一场是考察英语（英译中），然后会再拓展地问一下专业课知识（比较考察对相关科目的理解）；另一场是专业课+个人介绍+项目/竞赛。
  • 北邮：bar一般，按团队面试，不发offer，需要继续参加预推免。
    以上就是我参加或了解的夏令营。

2.预推免
由于东南信科今年结果出的比较晚，所以预推免报名期间，我投入的心思不多，冲着学校title以及想要旅游的心思（bushi 报了以下几所：

学校 入营 offer 备注
北大软微 否 / 没推荐信
中科院自动化所 否 /
中科大六系 否 / 动态审核，报名太晚
浙大工程师学院 是 未参加 移动智慧物联网项目
南大电院 是 未参加
南开光电 — / 通过网申，没推荐信
天大电信 是 未参加

预推免我报了很多，南开天大这两所仅有尝试的意味（名额已及时释放）。华五报了三所，科大由于报的太晚了（9.20截止，但其实9.12就出了一批入营），所以我也很无语。南大电院和浙大工院能进我多多少少能猜到，说实话我还是很想去这两所的。不过南大电院学科实力确实一般，预推免也很难找到强导，遂早早放弃。

浙大工院是9.19开奖，但是呢，那一天下午先是东南招生老师给打了电话问去不去，我当时等工院等的很烦了，并且东南这边也联系了一位很好的老师，就直接答应了，并且跟联系的导师进一步确认（已经聊得比较深入了），然后啊，我刚跟导师和师兄开了个小会，10分钟后浙大给发了复试通知…

综合考虑了一下，东南信息类的实力还是很强的，相关资源也很丰富，我也很喜欢南京这座城，并且导师人真的很好，除非我以后摆烂考公可能会不如上面几所有优势，其他都没啥问题。这样想，确实也没必要再去浙大折腾一番了。

四、总结

由于个人性格的原因，在夏令营阶段拿到满意的offer后，整个人都处于比较摆烂的状态，因此预推免一个都没参加。如果比较在意学校title的话，建议预推免还是多试试，不要像我一样直接躺平。

不过细想下来，在保研的整个过程中，大部分人难以兼顾学校title、专业实力、导师水平，想要做到面面俱到是非常困难的。因此即使没能去华五这一层次的高校，我也并没有那么遗憾。

最后，结合自身以及周围同学的经历，针对我自身的bg浅浅列举几所可以考虑的学校（信通方向）：

  • 清深
  • 上交电院直博
  • 浙大信电直博
  • 浙大工程师学院（项目制，科研勿进）
  • 中科大六系
  • 复旦信科
  • 南大电院
  • 哈工大深圳

上面列举的基本只能直博/专硕，我也是因为对这有顾虑，所以更多地考虑了如下通信强校/研究所：

  • 东南信科（求稳可以像我一样选择专硕，培养上无区别）
  • 北航、北理（偏军工）
  • 中科院对口强所（跟高校的氛围不同，慎重考虑）

篇幅所限，其他的不再列举，仅供参考。写下这篇经验贴更多的是对自己保研历程的一个回顾，如果有后来者看到，也希望能帮到你们，大家都有光明的未来~
